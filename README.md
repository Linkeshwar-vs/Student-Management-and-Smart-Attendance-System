# Student Management and Smart Attendance System

This project implements an automated attendance and student support platform that integrates multiple modern authentication technologies. The system was developed as part of an academic project.

---

## Objective

- Provide a secure and automated attendance system using multiple verification methods.
- Enable teachers to manage attendance sessions and student records efficiently.

---

## Key Features

| Category | Functionality |
|----------|---------------|
| Attendance Modes | Manual marking, QR-based (static and rotating), facial recognition with spoof detection, geolocation-based attendance |
| Session Management | Teacher can create and activate sessions, modify attendance with logs for audit tracking |
| Study Materials | Teachers can upload PDFs, images, or other resources |
| Student Dashboard | Timetable access, free slot identification, recorded attendance and quizzes |

---

## Technologies Used

- React.js, Flask, PostgreSQL
- Python
- Face Recognition (InsightFace) + Custom Anti-Spoofing Model
- Geolocation Services (GPS)
